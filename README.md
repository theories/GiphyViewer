# GiphyViewer

Search the Giphy API in iOS

##### Features

* Search Giphy.com using Giphy API
* Support for Giphy Trending data

##### Requirements

* Cocoapods v 1.5.0+
* XCode 9.2+

##### Installation

1. Clone the repository 
2. Run `pod install` at command line
3. Open **GiphyViewer.xcworkspace** in XCode
